# Github-User-Activity
A simple command line interface (CLI) to fetch the recent activity of a GitHub user and display it in the terminal.

https://roadmap.sh/projects/github-user-activity

# Requirements:
1. application should run from the command line
2. accept github username as argument
3. fetch the user's recent activity using github api
4. display it in the terminal

5. handle errors gracefully, such as invalid usernames or api failures
6. do not use any external libraries or frameworks to fetch the github activity
